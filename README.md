**Indian AI Travel Planner** — a Next.js web app where users enter a origin, destination, dates, budget, and preferences. An AI agent (Gemini) autonomously searches real flight/hotel/activity prices, checks weather, and produces a day-by-day itinerary guaranteed to fit within the user's budget.



following are requirements:
axios 
leaflet  
react-leaflet 
mathjs 
react-markdown 
install using `npm i <lib-name>`


.env file required to be added in base folder with 
GEMINI_API_KEY
and
SERPER_API_KEY
